ROSMOD GUI and Tool-SuiteÂ¶

This documentation covers the background, theory, and results for ROSMOD, which is designed as a modeling, generation, deployment, and management toolsuite for distributed CPS applications using ROS as the communications middleware.

ROSMOD is hosted as a development toolsuite for distributed CPS at rosmod, and documentation for using it can be found in the section Using ROSMOD.
